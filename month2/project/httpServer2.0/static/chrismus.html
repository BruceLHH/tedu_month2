圣诞节前备战断网！俄罗斯举行首次国家级断网演习，蠢or精？

新智元

发布时间：12-2618:09北京中经智元广告有限公司

【新 智元导读】俄罗斯政府宣布，已完成一系列测试，成功地使该国脱离了全球互联网，从而使俄罗斯的RuNet成为全球最大的内部网。此举引发专家担忧和网友热 议：这意味着人们将无法就本国正在发生的事情进行对话，而陷入自己的泡沫中。戳右边链接上 新智元小程序 了解更多！

俄罗斯政府宣布，俄罗斯已经成功测试了全球互联网的替代品。

根据通信部的说法，所涉及测试的详细信息尚不明确，但是普通用户没有注意到任何变化。现在将结果提交俄罗斯总统普京。

测试从上周开始进行了多天，涉及俄罗斯政府机构、当地互联网服务提供商和俄罗斯当地互联网公司。目的是测试俄罗斯的国家互联网基础设施（在俄罗斯内部称为RuNet）是否可以在不访问全球DNS系统和外部互联网的情况下运行。

互联网流量在内部进行了重新路由，从而使俄罗斯的RuNet成为世界上最大的内部网。

多年准备终于成功断网，俄罗斯法律允许以“国家安全”为由切断网络

俄罗斯政府没有透露有关测试及其系统组成的任何技术细节，只是说政府测试了几种断开连接的方案，包括模拟遭到来自国外敌对势力网络攻击的方案。

俄罗斯通信部副部长Alexei Sokolov说：“事实证明，总的来说，当局和电信运营商都准备好有效应对可能的风险和威胁，并确保俄罗斯的互联网和统一电信网络正常运行。”

其实，为了这项测试，俄罗斯政府已经准备了许多年，包括立法制定以及对俄罗斯本地互联网基础设施进行物理改造。

这项测试原定于今年4月进行，但被推迟到今年秋天，以使克里姆林宫有更多时间通过相应的法律。

这项被称为《互联网主权法》的法律赋予俄罗斯政府以“国家安全”为由，不加任何解释就可以随意将该国与互联网其他部分断开的权力。

为此，该法律要求所有本地互联网服务提供商在俄罗斯通信部的管理下，通过战略性阻塞点重新路由所有互联网流量。

一些专家仍对某些国家切断互联网的趋势感到担忧。萨里大学的计算机科学家Alan Woodward教授说：“可悲的是，俄罗斯的行进方向只是朝向互联网日益崩溃的又一步。”

“这意味着人们将无法就本国正在发生的事情进行对话，他们将陷入自己的泡沫之中。”

网络和连接越多、控制访问就越困难，国内互联网如何运作？

这项举措包括限制俄罗斯版本的互联网与全球互联网的连接点，让政府对其公民可以访问的内容有更多的控制权。

Woodward教授解释说：“就像大型公司一样，这将有效地让互联网服务提供商（ISP）和电信公司将其境内的互联网配置为一个巨大的内部网。”

那么，政府将如何建立某些人所谓的“sovereign Runet”呢？

各国通过海底电缆或“节点”接收外国网络服务，在这些连接点上，数据传输到其他国家的通信网络或从其他国家的通信网络传输。这些将需要被阻止或至少受到监管。

这将需要国内互联网服务提供商的合作，如果只有少数国有企业参与其中，这将更容易实现。一个国家拥有的网络和连接越多，控制访问就越困难。

那么，俄罗斯将需要创建替代系统。

在伊朗，国家信息网络允许访问网络服务，同时管制网络上的所有内容并限制外部信息。它由伊朗国有电信公司运营。

有效地将所有互联网接入转变为政府控制的“围墙花园”的好处之一是，通常用于规避数据块的虚拟专用网络（VPN）无法正常工作。

俄罗斯计划创建自己的维基百科，政客们通过了一项法案，禁止销售未预装俄罗斯软件的智能手机。

存在技术挑战

一位专家警告说，该政策可以帮助俄罗斯压制言论自由，但他补充说，该政策能否成功还没有定论。

New America网络安全政策研究员Justin Sherman表示：“过去，俄罗斯政府在尝试增加在线控制时遇到了技术挑战，例如在很大程度上未能成功阻止俄罗斯人访问加密消息App Telegram。”

“然而，如果没有更多关于这次测试的信息，很难准确评估俄罗斯在通往可隔离的国内互联网的道路上取得了多大进展。"

“在商业方面，俄罗斯将获得多少国内外的回击还有待观察。”

网友热议：应该是互联网成功的切断了俄罗斯

断开互联网？看到这消息的人大部分都蒙了，还能这么操作？这一新闻在Reddit上引发热议。

有网友嘲笑道：应该是互联网成功的切断了俄罗斯。

当然，并非所有网友都持嘲讽态度。有人就认为这是在积极防御，实际上，俄罗斯国防部很聪明地制定了该计划，因为下次战争将通过对基础设施的网络攻击进行大规模的战斗，那么就有必要提前预知如果网络被中断会发生什么。这次演习让俄罗斯看清了自己的能力，不可思议！

也有人认为这样做是为了为将来备战，这样俄罗斯可以攻击敌人的网络基础设施，而他们自己则可以像隔离病毒一样，切断和被攻击者的有害联系，从而保证自己的安全。

有的网友认为这也会在某种程度上使他们的基础架构更具弹性，通过重定向/控制DNS和所有路由（BGP）之类的内容，就不容易受到延迟或者中断。

有人盛赞此举，称这是俄罗斯有史以来做过的最聪明的举措之一！他非常认可这种方式，认为其能够有效地阻止任何外国网络攻击，这项新技术的防御潜力是巨大的。

有人担心长时间后会出现一些严重的问题，例如中央DNS记录，过期的证书，许可证检查，恶意软件扫描程序和软件更新。此外，缺少云访问将使智能手机和平板电脑几乎无用。
